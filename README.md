maven_sample
===============

## 概要
Javaで記述したソフトウェアを単体テストするための勉強用リポジトリ

## 使っているもの
- IDE
    - Eclipse
- ビルドツール
    - Maven
- テストライブラリ
    - JUnit
    - DbUnit
    - Mockito
    - Selenium
